# TikTok Takipçi Temizleme Aracı 🧹

Bu basit JavaScript kodu, TikTok hesabınızdaki takipçileri tarayıcı konsolu üzerinden toplu bir şekilde temizlemenize yardımcı olur.

## 🚀 Nasıl Kullanılır?

1.  **TikTok** web sitesine giriş yapın.
2.  Profilinizden **Takipçi** (Follower) listenizi açın.
3.  Sayfada `F12` tuşuna basarak (veya sağ tıklayıp İncele diyerek) **Konsol (Console)** sekmesine gelin.
4.  Bu depodaki kodu kopyalayıp konsola yapıştırın.
5.  `Enter` tuşuna basarak işlemi başlatın.

## ⚠️ Uyarı
Bu araç eğitim amaçlıdır. Çok hızlı işlem yapmak veya çok fazla takipçiyi aynı anda silmek TikTok hesabınızın geçici olarak kısıtlanmasına neden olabilir. Kullanırken sorumluluk size aittir.

## 🛠️ Teknolojiler
* **JavaScript** (Tarayıcı Konsolu / Browser Console)
